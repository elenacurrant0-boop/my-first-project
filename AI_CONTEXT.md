## Правила разработки (обязательно соблюдать)

- Каждая страница — **один самодостаточный HTML-файл**. Весь CSS и JS внутри него.
- **Никаких внешних библиотек** (Bootstrap, Tailwind, jQuery). Исключение — Google Fonts.
- **Дизайн:** тёмно-синий фон (~`#0a1628`), светлый текст (`#f0f4ff`), акцент золотой/оранжевый (`#f0a500`)
- **Адаптивность** от 320px: flexbox/grid + медиазапросы, шрифты через `clamp()`
- **Стиль текста:** простой и дружеский, без канцелярита, короткие предложения
- **Перед созданием нового файла** — описать план, получить подтверждение, потом писать код
- При редактировании — сначала прочитать файл, понять контекст, потом вносить изменения
