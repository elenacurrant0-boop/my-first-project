# Проект: Личный бренд + клиентские лендинги (курортная тема)

## Что это за проект

Репозиторий совмещает два направления: личный сайт-визитка **Елены Смородиной, эксперта по курортам КМВ**, и клиентские веб-материалы для объектов санаторно-курортной отрасли. Все страницы написаны на чистом HTML/CSS без фреймворков — каждая является самодостаточным файлом. Проект находится в активной разработке и постепенно превращается в полноценное портфолио веб-разработчика в нише курортного бизнеса.

## Стиль дизайна

- **Подход:** чистый HTML + CSS, без фреймворков, каждая страница — отдельный файл
- **Анимации:** CSS-анимации (солнечные лучи, облака, shimmer-эффекты, parallax)
- **Типографика:** Cormorant Garamond (элегантность), Inter / Onest (читаемость), Montserrat (заголовки)
- **Палитры по проектам:**
  - Визитка: небесно-синяя (#003278 → #5bc5f8)
  - Золотая Русь: зелёный (#1e5c3a) + золото (#c8961a)
  - Aqva SPA: морской (#00AACC) + коралловый (#FF6B6B) + песочный
- **Общий характер:** курортная эстетика, природные образы, премиальный вид при минималистичной структуре
